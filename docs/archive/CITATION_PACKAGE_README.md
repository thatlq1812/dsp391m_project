## â“ FAQ

**Q: What if AI can't find a paper?**
A: Mark as "NOT FOUND" in template. We can proceed with 10/15 minimum.

**Q: Multiple versions exist - which to use?**
A: Official conference/journal > arXiv > workshop > preprint

**Q: How to verify BibTeX compiles?**
A: Use bibcheck.org or test with LaTeX compiler

**Q: Can I skip supporting papers?**
A: Yes, if time is limited. Focus on Critical (1-7) and Important (8-12).

**Q: What if citation details don't match our notes?**
A: Trust official source. Update our notes if needed. Flag discrepancies.
